## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/abhilashindulkar/abhilashindulkar.git

# Navigate to docs
cd abhilashindulkar/docs

# Open index.html in browser or use a local server
python -m http.server 8000
```

Visit `http://localhost:8000`
